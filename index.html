angular-snippets

Angular direct-ives to display code.

  • Use Prism syntax highlighter
  • Fully configurable by templates and css
  • Support theme

Example


<snippets path="snippets/demo" files="['index.html', 'app.js', 'missing.css']">
  <pane snippet="{name: 'run', icon: 'fa-cog'}">
    <iframe src="snippets/demo/index.html"></iframe>
  </pane>
</snippets>

Getting started

Dependencies

  • AngularJS
  • Prism

installation

You can either download the latest version on the github repository or use Bower to install it automatically: 
bower install angular-snippets
Bower will also install its dependencies. Load the packages into your HTML page: 

<link rel="stylesheet" type="text/css" href="path/to/prism/prism.css">
<script src="path/to/prism/prism.js"></script>
<script src="path/to/angular-snippets.min.js"></script>
You can use official themes, if so, you also need to load it. 

<link rel="stylesheet" type="text/css" href="path/to/theme/bootstrap/style.css">
<script src="path/to/theme/bootstrap/theme.js"></script>

API

Provider

snippetsProvider

If you don't use an official theme or want to extend it, you can use snippetsProvider to configure the module: 

app.config(function(snippetsProvider) {
  snippetsProvider.configure({
    cls: {
      missing: 'error'
    }
    tabs: {
      //template: '...',
      templateUrl: 'partials/tabs.html'
    },
    pane: {
      //template: '...',
      templateUrl: 'partials/pane.html'
    }
  });
});

Directive

snippets

snippets is the main directive.
Basically, it accepts an array of filenames, download these files using $templateRequest, highlight it using Prism and display them. path to prepend on each filename,
e.g. path="snippets/" Array of string. File names list to display,
e.g. files="['index.html', 'script.js']"

pane

pane is a file element to display.
Pane elements provided by ng-repeat on snippets.files provide in the scope: File properties Where snippet contains: File name Prism type, e.g. javascript File content Class to add (if file is not available, add default cls.missing class) Tab has to be disabled (if file is not available)
